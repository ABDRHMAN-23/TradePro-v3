# ⚡ TradePro v3.0 — أداة المقاول الذكي

تطبيق PWA كامل لإدارة أعمال المقاولين مع ذكاء اصطناعي Gemini (آمن عبر سيرفر وسيط).

## 🔧 الإصلاحات في هذه النسخة

| المشكلة | الحل |
|---------|------|
| مؤقت المهمة غير مرغوب | ✅ تم حذفه بالكامل |
| المواد لا تظهر اسمها/سعرها في عرض السعر | ✅ إصلاح كامل — كل البيانات تظهر بوضوح |
| إضافة عميل معقدة | ✅ نموذج مبسط (اسم + جوال فقط مطلوبين) + إضافة سريعة من داخل نموذج المهمة |
| لا يوجد دليل استخدام | ✅ زر "❓ دليل الاستخدام" في الشريط الجانبي يشرح كل ميزة بالتفصيل |
| البحث يتوقف عند كل حرف | ✅ تم إصلاح باستخدام `defaultValue` بدل `value` controlled |
| المساعد الذكي "أحمق" | ✅ **دمج Gemini AI الحقيقي عبر سيرفر آمن** + نظام احتياطي محلي ذكي |
| الكتالوج بلا تصنيفات | ✅ تصنيفات تلقائية (كابلات، قواطع، صمامات...) + بحث + **كتالوج جاهز محمّل تلقائياً حسب مهنتك** |
| تصدير CSV "أحمق" | ✅ تصدير ذكي بعناوين عربية كاملة، حسابات صحيحة، رؤوس أعمدة احترافية |

## ✨ الميزات الجديدة المُحتكَرة

- **🤖 Gemini AI حقيقي وآمن**: المفتاح يعيش فقط على السيرفر (متغير بيئة)، أبداً لا يصل للمتصفح أو المستخدم — ويحلل بياناتك الحقيقية
- **📦 كتالوج جاهز بالضغطة**: اضغط "تحميل الكتالوج الجاهز" ويُحمَّل 15-20 مادة شائعة لمهنتك تلقائياً مع الأسعار
- **📝 عروض الأسعار → فواتير**: نظام كامل لإرسال عرض سعر، تحويله لفاتورة بضغطة
- **📅 تقويم أسبوعي مرئي**
- **👷 إدارة الفريق**
- **❓ دليل استخدام مدمج** يشرح كل خطوة

## 🔒 الذكاء الاصطناعي عبر Gemini (آمن بالكامل)

تم تغيير معمارية الذكاء الاصطناعي بالكامل لأسباب أمنية:

**كيف يعمل الآن:**
```
المتصفح (TradePro.jsx)  --[رسالة بدون مفتاح]-->  server.js  --[المفتاح هنا فقط]-->  Gemini API
```

- **المفتاح لا يظهر أبداً في كود الواجهة** ولا في أدوات المطور (F12) ولا في أي ملف يصل للمستخدم
- المفتاح يُقرأ من متغير بيئة `GEMINI_API_KEY` على السيرفر فقط
- إن لم يكن السيرفر يعمل (أو بدون مفتاح)، يتحول التطبيق تلقائياً للمساعد المحلي الذكي البديل دون أي خطأ يظهر للمستخدم

### إعداد المفتاح (مرة واحدة، على السيرفر فقط)

```bash
cp .env.example .env
# افتح .env وضع مفتاحك:
# GEMINI_API_KEY=AIzaSy...

npm install
npm run server
```

احصل على مفتاح Gemini من: https://aistudio.google.com/app/apikey

### ⚠️ تنبيه أمني مهم

إن كنت قد شاركت مفتاح Gemini سابقاً في أي محادثة أو كود عام، **اعتبره مكشوفاً وأبطِله فوراً** من Google AI Studio، ثم أصدر مفتاحاً جديداً واستخدمه فقط داخل `.env` (وتأكد أن `.env` مُدرَج في `.gitignore` — وهو كذلك بالفعل في هذا المشروع).



```bash
npx create-next-app@latest tradepro --app
cd tradepro
cp TradePro.jsx src/app/page.tsx
cp manifest.json service-worker.js public/
npm run dev
```
